# Learn Docker With Me Like Never Before 🚀

Welcome to the ultimate hands-on Docker learning repository! Here, I share everything I teach in my Docker classes, making it easy for you to follow along and master Docker step by step.

## 📚 What We've Covered So Far

I've conducted **3 classes** so far, and we've explored the following topics:

- **Docker Introduction:** What is Docker and why use it?
- **Virtual Machines vs Containers:** Understanding the differences.
- **Containers:** What are containers and how do they work?
- **Images:** What are Docker images and how are they used?
- **Dockerfile:** Introduction to Dockerfile and its purpose.
- **Dockerfile Instructions:** Detailed explanation of each instruction.
- **Docker Hub & Docker Registry:** Storing and sharing images.
- **Creating Containers:** How to launch and manage containers.
- **Creating Images:** Building images from Dockerfiles.
- **Creating Custom Images:** Personalizing your own images.
- **Volumes:** What are volumes and why are they important?
- **Sharing Volumes:** 
  - Container-to-Container sharing
  - Host-to-Container sharing

## 📝 How to Use This Repo

- Each folder corresponds to a class or topic.
- Code samples, Dockerfiles, and notes are included for every concept.
- Follow along with the class notes or try the examples yourself!
